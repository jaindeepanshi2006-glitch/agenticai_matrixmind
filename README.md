# CareerHub - AI-Powered Job & Internship Platform Frontend

A modern, responsive web frontend for an Intershala-like career platform with AI-powered resume analysis and job recommendations.

## 🚀 Features Implemented

✅ Professional Navigation Bar with all Intershala-like buttons
✅ Job Listing Page with advanced filters and search
✅ Internship Listing Page with duration and stipend filters
✅ AI Chatbot Interface for resume analysis and recommendations
✅ User Profile Page with experience, skills, and resume management
✅ Application Tracking Page with status management
✅ Responsive Design (desktop, tablet, mobile)
✅ Professional CSS Styling with animations

## 📁 Files Created

- **index.html** - Home/Landing page with featured jobs & internships
- **pages/jobs.html** - Jobs listing with advanced filtering
- **pages/internships.html** - Internships listing with custom filters
- **pages/chatbot.html** - AI chatbot for resume analysis
- **pages/profile.html** - User profile with tabs
- **pages/applications.html** - Application tracking
- **css/style.css** - Complete responsive styling (1000+ lines)
- **js/script.js** - JavaScript functionality

## 🎯 Quick Start

1. Open `index.html` in your web browser
2. Navigate using the top navbar
3. Try different features on each page
4. Responsive on all devices

## 📱 Browser Support

Chrome, Firefox, Safari, Edge - Latest versions
